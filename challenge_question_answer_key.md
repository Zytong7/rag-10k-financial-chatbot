# Challenge Question Answer Key

## Question

Based on the companies' 2025 10-K reports, rank Microsoft Intelligent Cloud,
Amazon Web Services (AWS), and Google Cloud from highest to lowest by fiscal
year 2025 operating margin.

## Standard answer

1. **Microsoft Intelligent Cloud — 42.0%**
   - Revenue: $106,265 million
   - Operating income: $44,589 million
   - Calculation: $44,589 / $106,265 × 100% = 42.0%
   - Source: Microsoft 2025 10-K, PDF page 69; Note 18 on PDF page 144.

2. **Amazon Web Services (AWS) — 35.4%**
   - Net sales: $128,725 million
   - Operating income: $45,606 million
   - Calculation: $45,606 / $128,725 × 100% = 35.4%
   - Source: Amazon 2025 10-K, Note 10, PDF page 110.

3. **Google Cloud — 23.7%**
   - Revenue: $58,705 million
   - Operating income: $13,910 million
   - Calculation: $13,910 / $58,705 × 100% = 23.7%
   - Source: Alphabet 2025 10-K, Note 15, PDF page 84.

## Final ranking

**Microsoft Intelligent Cloud > AWS > Google Cloud**

## Important comparability note

Microsoft's fiscal year 2025 ended June 30, 2025, while Amazon's and
Alphabet's fiscal years ended December 31, 2025. The calculation is still a
valid comparison of each company's reported FY2025 segment operating margin,
but the reporting periods do not cover the same calendar months.
